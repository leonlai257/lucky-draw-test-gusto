

## Installation

```bash
$ npm install
```

## Docker setup

```
docker run --name lucky-draw-test -e 'ACCEPT_EULA=Y' -e 'SA_PASSWORD=luckydraw123' -p 1433:1433 -d mcr.microsoft.com/mssql/server:2019-latest
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
